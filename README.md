# Prime Event — Workshop Registration

Multi-step ticket registration with TikTok Shop checkout and digital ticket delivery. Vanilla HTML/CSS/JS — zero dependencies, runs from any static host.

## Live Demo

- Workshop: https://workshop-form-01.netlify.app/workshop/
- Fitness: https://workshop-form-01.netlify.app/fitness/

## Structure

```
RegForm/
  fitness/index.html     ← fitness client registration
  workshop/index.html    ← event workshop signup (3-step + TikTok Shop)
  workshop/ticket.html   ← digital ticket with print/save-as-PDF
```

## Flow

1. Select ticket tier & quantity
2. Enter name, email, phone
3. Pay via TikTok Shop → tap "I've Paid — Get My Ticket"
4. Print or save your digital pass as PDF

Built with vanilla HTML/CSS/JS. All prices in ₦.
